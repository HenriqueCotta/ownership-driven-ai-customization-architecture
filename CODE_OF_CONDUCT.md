# Code Of Conduct

## Our Standard

This project is intended to be a respectful, practical, and technically serious place to collaborate.

Participants are expected to:

- assume good intent while still being direct about technical concerns
- critique ideas, structures, and trade-offs without attacking people
- help newcomers understand the model without gatekeeping
- keep discussion grounded in evidence, examples, and maintainability

## Unacceptable Behavior

Examples of unacceptable behavior include:

- harassment, intimidation, or personal attacks
- mocking basic questions
- deliberately hostile or demeaning language
- bad-faith disruption of issues, pull requests, or review discussions
- publishing private information without consent

## Scope

This code of conduct applies to:

- issues
- pull requests
- discussions
- review comments
- other public community interactions in this repository

## Enforcement

Maintainers may remove, edit, or reject contributions or comments that violate this code of conduct.

## Reporting

If you see a problem, open a private channel with the maintainers if available. If no private channel exists yet, open a minimal public issue asking for one without posting sensitive details.
