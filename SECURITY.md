# Security Policy

This repository is mainly documentation, templates, and starter-kit material.

Security concerns may still matter when they affect:

- starter-kit examples
- workflow files
- automation scripts
- guidance that could encourage unsafe defaults

## Reporting A Concern

If you find a security-related problem, please avoid posting sensitive details in a public issue.

Instead, open a minimal issue asking for a private reporting path, or use private reporting if the repository enables it.

## What To Include

- the affected file or path
- why the issue matters
- the likely impact
- a minimal explanation that is safe to share
