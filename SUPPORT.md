# Support

Use the following channels:

- bug or correctness problem
  - open an issue
- proposal for the architecture, starter kit, or templates
  - open an issue or discussion if discussions are enabled
- question about adoption in another repository
  - open an issue and describe your repository shape and constraints

When asking for help, include:

- a short description of your repository or use case
- the problem you are trying to solve
- anything you already suspect may be relevant
